## Laundry Service Webpage

Project Description

This project is a simple webpage created using HTML and CSS.
It shows a basic layout of a laundry service website.

# The page contains:

-> A navigation bar
-> A hero section with text and an image
-> A button to book a service

Display Flex is not used in this project. The layout is created using "display: inline-block".

# Sections in the Webpage

Navbar

The navigation bar contains:

-> Logo on the left
-> Navigation links in the center (Home, Services, About Us, Contact Us)
-> Username button on the right

# Hero Section

The hero section has two parts:

-> Left side: Heading, description, and a button
-> Right side: Image related to laundry services

Viewport height is used so that the hero section fits the screen and does not create scroll.

# Files

2. Open the "index.html" file in a web browser.
